# flutter_webtoon_app

A new Webtoon project by Flutter.

## Getting Started
# 웹툰 클론 프로젝트
- 사용 기술 
  1. 데이터 불러오기
  2. 인터넷에서 패키지 다운 받아 사용하기
  3. 스마트폰의 API 사용하기 - API에서 받을 응답 : ** 제목, URL, ID **
  4. 스마트폰에 데이터 저장하기
  5. 캐러셀 사용
